# Northwind Sales Analytics Project

## Project Overview

This project analyzes the Northwind Traders dataset using Excel, SQL, Power BI, and DAX to generate business insights related to sales performance, customer behavior, product analysis, supplier operations, employee contribution, and regional sales distribution.

The project includes interactive dashboards, KPI analysis, EDA techniques, and business reporting to support data-driven decision-making.

---

## Tools & Technologies Used

- **Excel** — Data Cleaning & Exploratory Data Analysis
- **SQL** — Data Querying & Business Analysis
- **Power BI** — Dashboard Development & Visualization
- **DAX** — KPI Measures & Calculated Metrics

---

## Dashboard Sections

- Executive Overview Dashboard
- Customer Analysis Dashboard
- Employee Analysis Dashboard
- Product & Sales Analysis Dashboard
- Supplier & Operations Dashboard

---

## Key Insights

- Total revenue exceeded **$1.35M** across **760 orders**
- Beverage and Dairy Products generated strong sales contribution
- Customer distribution was concentrated across North America and Europe
- Average shipping duration was approximately **8.65 days**
- Revenue trends showed strong growth during peak sales periods
- Employee contribution analysis highlighted operational performance insights
